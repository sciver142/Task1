# ARK GitHub Training Repository

Welcome, trainees! This is the official repository where all your summer tasks will be hosted.

The goal of these tasks is to progressively introduce you to the tools, workflows, and concepts you will use throughout your tenure at **Aerial Robotics Kharagpur (ARK)**. Each task is designed to build on the previous one, so make sure you engage with each of them seriously.

## Repository Structure

```
ark_github/
├── TASK 1/    ← Git and GitHub
├── TASK 2/    ← (coming soon)
└── ...
```
New tasks will be released each week based on your progress and responses.

## Important Notes

- **Document everything.** Get into the habit of maintaining clear documentation for all your work at ARK, it will be used to evaluate your progress during review meets.
- **Ask questions.** If you are stuck or confused at any point, reach out to any of us. Do not hesitate.

We are excited to have you on board. Let's build something great together.
